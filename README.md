# Simple EVE Echoes discord bot

Current supported modules:
- market

## Installation

```bash
npm i
```

## Config

- Copy ```env.example``` to ```.env```. 
- Set your ```TOKEN``` in ```.env``` file. 

## Usage

```
npm start
```

## Commands

```
!market Item_name
```

Example:
```
!market stab
```

The output will be current Stabber price on the market.
